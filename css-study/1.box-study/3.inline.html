CSS의 inline 을 학습하기 위해 작성하는 HTML. inline 의 경우 공간이 부족할 경우 다음 줄로 넘어가는 특징을 가지고 있는데 쉽게 말해서 글을 작성할 때 공간이 없는 경우 자동으로 다음 줄로 넘어가서 계속 작성되는 형태를 의미한다고 생각하면 이해하기 쉽다. 모든 HTML 요소는 display 값을 가지게 되는데 span, a, strong 태그의 경우 display inline 요소 중 하나이다. 1. inline의 경우 margin, padding 속성의 좌우 간격(left, right)만 사용할 수 있고 top, bottom은 사용할 수 없다. 2. block은 한 줄을 다 차지하는 반면 inline은 위에서 언급한 바와 같이 태그, 태그1, 태그2 줄 바꿈 없이 보이게 된다.
